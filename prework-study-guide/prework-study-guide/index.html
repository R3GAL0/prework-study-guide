Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: enters the main branch
  • git pull origin main: updates the main(local) branch with the current main(github branch)

      • Creating a github pull request - from the branch enter the following commands:
    1. git add -A
    2. git commit -m "notes"
    3. git pull origin main ((to see that the main local branch is updated))
    4. git push origin file/subfile ((pushes the changes to github))
